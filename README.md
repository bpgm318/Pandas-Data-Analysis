# Pandas-Data-Analysis

This analysis pulls data from 2 datasets
  - mb.csv - this is the mobile data usage of over 2800 users for the period of April 2025 to March 2026
  - minutes.csv - this is the mobile voice usage of the same 2800 users for the period of April 2025 to March 2026

Pandas and Matplotlib are the libraries used to clean the data and perform exploratory data analysis. At the end 2 new summary csv files are created which contain some of the  metrics analyzed during the analysis.
